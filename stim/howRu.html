How This Works

Many people experience emotions as physical sensations rather than named feelings. This is especially common for neurodivergent individuals. This tool helps translate those physical experiences into emotional language.

Select the physical sensations you're experiencing and we'll match them to possible emotions based on the Type-0 Emotion Translation Matrix.

Emergency Override Activated

Your physical sensations indicate possible system overwhelm. Consider:

  • Reducing sensory input
  • Finding a quiet space
  • Using simple communication: "I'm experiencing system overload"
  • Basic self-care: water, comfortable position, deep breathing
Understanding Your Physical-Emotional Connection

What you're experiencing is valid and recognized. Here's what might help:

  • Keep a sensation journal: Note physical feelings and what was happening
  • Use this tool regularly: Build connections between sensations and emotions
  • Be patient with yourself: This is a learning process
  • Remember intensity matters: Mild chest tightness might be worry, severe might be panic
  • Context clues help: What happened right before the sensations started?
