This guide provides a clean, list-only reference of the essential VoiceOver gestures for your iPhone. Practice these one at a time to master VoiceOver navigation.

Core Concept

Single-tap to select an item. Double-tap anywhere on the screen to activate the selected item.

This fundamental principle is key to understanding how VoiceOver transforms your interaction with the iPhone.

Essential Gestures

Start with these basic gestures. Master selecting and activating before moving to more advanced techniques.

Gesture Description
Tap Selects an item and VoiceOver speaks its name.
Double-tap Activates the selected item (e.g., opens an app, presses a button).
Swipe Left/Right Moves selection to the previous/next item.
Flick Left/Right Same as swipe; moves to the previous/next item.

Navigation & Reading

These gestures help you navigate content and control speech.

Gesture Description
Drag (Explore by Touch) Lightly drag your finger around the screen. VoiceOver will read every element you touch.
Two-finger tap Stops current speech.
Two-finger double-tap Starts/Stops the current action (e.g., play/pause music, answer/end a call).
Three-finger swipe up/down Scrolls the screen one page at a time.
Three-finger swipe left/right Changes pages (e.g., on the Home Screen).
Three-finger tap Toggles the "Speech Mute" (silences VoiceOver temporarily).

Advanced Power Gestures

Once you've mastered the basics, these gestures will dramatically improve your efficiency.

Gesture Description
Two-finger scrub ("Z" gesture) Move two fingers back and forth horizontally three times quickly. This acts as the "Go Back" or "Home" command.
Rotor Turn two fingers on the screen as if you're rotating a dial. This changes what happens when you swipe up/down.
Swipe Up/Down (with Rotor) After using the Rotor to select an option (like "Headings"), swiping up/down will jump between those elements.

Emergency Gesture

If you ever get stuck or overwhelmed, use this to quickly exit VoiceOver.

"Hey Siri, turn off VoiceOver."

The easiest way to exit if you get stuck. This voice command will immediately disable VoiceOver.

Practice Strategy

Start with gestures 1-4. Master selecting and activating. Then move to 5-7. The rest will come with time.

Remember to practice in a quiet environment and be patient with yourself as you develop new muscle memory.
