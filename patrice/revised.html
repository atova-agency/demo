Philosophie Tarifaire Révisée

La portée précédente était trop extensive et coûteuse pour un engagement initial. Cette proposition révisée garde le MVP intact à €4,500 mais décompose le développement ultérieur en packages optionnels plus petits et abordables qui peuvent être achetés individuellement selon les besoins évolutifs et le budget du client.

  • MVP (Phase 1) : €4,500 - Système de base non-négociable
  • Packages Futurs : €1,000 - €1,500 chacun - Modules optionnels et autonomes

MVP - Système de Base (€4,500)

20 Septembre - Mi-Novembre 2025

Cette phase reste inchangée et est essentielle. Elle livre le système fondamental qui remplace immédiatement les processus manuels.

Fonctionnalités Incluses

  • Authentification sécurisée avec gestion des rôles utilisateurs
  • Gestion complète d'inventaire (CRUD, Recherche, Filtres, Alertes stock bas)
  • Module de facturation avec génération PDF automatique
  • Rapports de base : Journal d'activité et volumes de vente
  • Installation sur serveur du client avec formation complète
  • Support technique inclus : 30 jours

Spécifications Techniques

  • Frontend : Next.js 15+ avec TypeScript
  • Backend : API Routes Next.js
  • Base de données : PostgreSQL 16
  • Authentification : Next-Auth.js avec RBAC
  • Styling : Tailwind CSS
  • Déploiement : Docker + Docker Compose
  • Formation : 2 heures incluses
